| 0x040  | uscratch | Scratch register for user trap handlers |
| 0x041  | uepc     | User exception program counter          |
| 0x042  | ucause   | User trap cause                         |
| 0x043  | utval    | User bad address or instruction         |
| 0x044  | uip      | User interrupt pending                  |
