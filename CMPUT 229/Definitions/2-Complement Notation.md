2. Invert Bits: 0101
3. Add 1: 0110
4. Convert 0110 to decimal: 6
5. Add correct sign: -6

3. Add 1: 0110 0101
4. Convert 0110 0101 to decimal: 101
5. Add correct sign: 1001 1011
